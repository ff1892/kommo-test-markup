# Тестовое задание на вёрстку

## Технологии

* HTML
* SCSS

## Запуск

`git clone git@github.com:ff1892/kommo-test-markup.git`  

`npm i`

**Режим разработки**: порт `3000`

`npm start`

**Билд**

`npm run build`
